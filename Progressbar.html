Progress Bars

Nest 2 divs, the outer div must use .progress class to create the full width empty space for progress bar to be filled with the inner div. The inner div must contain: class="progress-bar" role="progressbar" aria-valuenow="70" aria-valuemin="0" aria-valuemax="100" style="width:70% values. This values fill the progress bar with color and %

Use a span with .sr-only class within to remove from the progress bar the visible percentage.
